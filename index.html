출퇴근 앱 사용 안내

위지원시스템즈의 출퇴근 관리는 모바일 앱으로 이루어집니다.
앱 설치 및 회원가입 후 이용해 주세요.

출퇴근 기록 방법

  • 출퇴근 시 ‘출퇴근’ 버튼을 눌러 기록해 주세요.
  • 출근지가 아닌 곳에서 출근하거나, 지각, 조퇴, 반차, 앱 오류 등의 사유로 인해 비정상적인 출퇴근이 발생할 경우 사유 입력 팝업이 표시됩니다. 해당 팝업에서 사유를 작성해 주세요.
  • 출근 후 외근을 가실 경우 ‘외근’ 버튼을 눌러 기록해 주세요.

앱 설치 안내

앱 설치 페이지: 여기를 눌러 설치 페이지로 이동하세요

안드로이드 사용자

  • 위 설치 페이지에서 앱을 다운로드하여 설치해 주세요.

아이폰 사용자

  • 애플 정책상 기기 등록 및 개발자 모드 설정이 필요합니다.
  • 아래 UUID 확인 페이지에서 UUID를 확인한 후, 최우수 과장님께 UUID와 함께 기기 등록을 요청해 주세요.
  • 만약 위 방법으로 UUID 확인이 어려우시면, 기기를 본사로 가져오신 후 최우수 과장님께 기기 등록을 요청해 주세요.
  • 기기 등록은 1~3일 정도 소요되며, 등록 완료 후 설치 페이지에서 앱을 설치하고 개발자 모드를 활성화하여 사용하시면 됩니다.
  • 방문하실 때 부재중일 수 있으므로 방문 전 미리 연락해 주시면 감사하겠습니다.
